アイコン・デコレート|Decorate-Icon


２つの画像を重ねるサービスです。
			使い方:
			1.背景画像を選択。
			2.重ねる画像を選択。
			3.合成画像をダウンロード。
			4.SNSのアイコンを設定！
1.背景画像|Base Image
2.重ねる画像|Cover Image
3.合成画像|Overlay Image
			Overlay images for SNS icon.

			usage:
			1.select Base Image.
			2.select Cover image.
			3.download image.
			4.set SNS icon!
